# Credit Risk Analysis & Loan Default Prediction

## Overview
This project analyzes loan application data to identify patterns related to borrower default behavior. The objective is to support credit decision-making using data-driven insights.

## Key Steps
- Data cleaning and preprocessing using Python and SQL
- Exploratory data analysis to identify risk patterns
- Predictive modeling to classify high-risk applicants
- Performance evaluation using classification metrics
- Dashboard creation (Power BI) to visualize risk trends and portfolio performance

## Tools Used
- Python (Pandas, Scikit-learn)
- SQL (SQLite)
- Power BI
- Streamlit (for demonstration)

## Business Impact
- Identifies high-risk borrowers
- Highlights key financial risk drivers
- Supports data-backed loan approval decisions
